Хотим скомпилить, используюя другую библиотеку. можно при компиляции указать параметр, где находятся *.class - файлы параметром -classpath
javac -classpath target src/Demo.java
можно сократить до -cp
javac -d target -cp target src/Demo.java
	
		Исходники и .class - файлы не удобно перекидывать, поэтому проще всего перекидывать jar-архивы. Запаковать бинарники в архив можно так:
	jar -cvf <имя jar-файла> <путь, где лежат файлы>
		-c create - создать jar-файл
		-v verbos - внести подробную информацию
		-f задать имя
	Запустить можно из архива вот так: java -cp <Имя jar-файла> <Имя класса>
	Просмотреть манифест jar-файла можно с помошью jar -tf <имя jar-файла>
	Если класс не стартует, можно зайти в манифест и прописать Main-Class:Demo
