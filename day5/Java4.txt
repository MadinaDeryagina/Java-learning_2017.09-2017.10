	Какие бывают модификаторы доступа:
	private
	public - может применяться ко всему к полю, к методу, к классу. 
	Что если мы промодифицируем класс, как private? Компилятор на верхнем уровне не даст модификатор private на класс, его можно сделать для вложенных классов.
	Если для класса не указать область видимости - по умолчанию у него будет модификатор package.
	package может применяться для класса, для методов, для полей. Что если нам хочется отнаследоваться от класса, но при этом находиться в другом пакете?
Для таких вещей есть модификаторв protected. Если class protected - он виден для классов из других пакетов.
