			Среда разработки.
Есть 3 основные среды разработки: IntellIDEA, Eclipse, NetBeans.
IntellIDEA - очень сильно ориентирована в код. Но если придется работать в фреймворками, то они больше заточены под Eclipse.
IntellIDEA распространяется в двух версиях: community edition - бесплатная версия, professional version - на месяц, потом будет говорить, плати денежку позяяя.
		Так как мы собираемся использовать maven, то надо не провтыкать и сказать, что это именно maven - проект.
		Проект будет собираться на сервере, который будет стягивать к себе все изменения и собирать проект.
		Наш maven - проект будет собираться на других машинах очень часто.
Чтобы открыть maven - проект:
		Open -> Выбираем папку с проектом maven.
		Если IntellIDEA не понимает, что проект mvn, папка src не подсвечивается синим, внутри там для нее просто файлы, тогда надо помочь идеи понять,
что там лежить именно файлы с исходниками, тогда мы нажимаем на папке проекта и выбираем marked directory as и там выбираем SourceRoot
		Нажмите 2 раза alt и на второе нажатие держать - справа появляется панель, там выбираем проект, если там пусто, нажимаем "+" и выбираем наш проект.
		
		Будем стараться кодить без мышки.
		
		Hot-keys:
		1. Чтобы быстро перкдлючаться по вьюшкам, называем alt+<цифра-номер вьюшки>
Если нет номер вьюшки: File-Settings-в поиске вводим keymap - там справа второй поиска, вводим например maven, нажимаем 2 раза и нажимаем сочетание клавиш,
и у нас создается hot-key. Имеет смысл вешать хоткеии на консоль, на мавен, на гит.
		2. Внизу слева есть кнопочка, которая позволяет увидеть окошки, где можно увидеть во вкладке VersionControl -> Log, там можно увидеть поток изменений,
при нажатии на коммит - видим, что изменилось за 1 коммит(какие файлы изменились и т.д.). Шифтом можно выбрать несколько коммитов и посмотреть, какие изменения
были сделаны за все эти коммиты.
		3. Чтобы добраться до любого функцианала IntellIDEA, можно ввести ctrl+shift+a - поиск по функционалу IntellIDEA.
		4. Есть еще более крутой поиск - 2 раза нажать shift, тогда появится поиск не только по действиям, но и можно будет провести поиск по всем своим классам

		Идея умеет сокращать, например какой нибудь psvm - public static void main.
		5. Нажимаем ctrl+пробел и получаем список всех снипетов.
		
Чтобы запустить:		
		1. Прежде чем запускать, надо решить с каким JRE запускать.
		2. Имя класса, которое запускаем.
		3. Надо определить classpath - это определяет maven
		4. При запуске иногда надо настроить ключи виртуальной консоли.
		5. Определить входные параметры, которые передаются в метод main.
		Все это можно определить в конфигурации запуска: Run - Edit configuration. Там можно определить ключи для java-машины, 
		
		Какие еще комбинации есть: иногда надо подвигать строки: ctrl+shift+<клавиши вверх-вниз>
		Списко всех хоткеев можно посмотреть лучше в гугле intellj idea hotkeys
		
		
		VCS - version control service
		VCS operation popup - там собраны все операции для контроля версий, есть commit, есть push
